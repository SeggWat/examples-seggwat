Static Site Example

SeggWat Widgets

Embed feedback and rating widgets on any static site. No build tools, no frameworks — just HTML.

Feedback Button

A floating button that opens a feedback modal. Supports screenshots, custom colors, positions, and languages. Look for the button on the right side of the screen.

Embed Code

<script defer
  src="https://seggwat.com/static/widgets/v1/seggwat-feedback.js"
  data-project-key="YOUR_PROJECT_KEY"
  data-button-color="#10b981"
  data-button-position="right-side"
  data-enable-screenshots="true">
</script>

Helpful Rating

A simple thumbs up/down widget. Place it inline on any page to ask "Was this helpful?"

Live Demo

Embed Code

<!-- Container for inline placement -->
<div id="helpful-widget"></div>

<script src="https://seggwat.com/static/widgets/v1/seggwat-helpful.js"
  data-project-key="YOUR_PROJECT_KEY"
  data-question="Was this page helpful?"
  data-show-comment-link="true"
  data-container="#helpful-widget">
</script>

Star Rating

Star or smiley rating widget. Supports 1-5 stars, 1-10 stars, 3-face smileys, and 5-face smileys.

Live Demo (Stars)

Live Demo (Smiley)

Embed Code

<!-- Star rating (default 1-5) -->
<div id="star-widget"></div>

<script src="https://seggwat.com/static/widgets/v1/seggwat-rating.js"
  data-project-key="YOUR_PROJECT_KEY"
  data-question="Rate your experience"
  data-container="#star-widget">
</script>

<!-- Smiley variant (3-face) -->
<div id="smiley-widget"></div>

<script src="https://seggwat.com/static/widgets/v1/seggwat-rating.js"
  data-project-key="YOUR_PROJECT_KEY"
  data-question="How do you feel?"
  data-style="smiley"
  data-faces="3"
  data-container="#smiley-widget">
</script>

Configuration Reference

Attribute Widget(s) Description Default
data-project-key All Project UUID (required) —
data-button-color Feedback, Helpful, Star, Contact Hex color for buttons/accents #2563eb
data-button-position Feedback, Contact Floating button layout bottom-right
data-language All UI language (en, de, sv, fr) Auto-detect
data-enable-screenshots Feedback Enable screenshot capture false
data-question Helpful, Star Custom question text Was this page helpful?
data-container Helpful, Star CSS selector for inline placement Script location
data-show-comment-link Helpful Show "Add comment" after negative rating false
data-style Star Rating style (stars or smiley) stars
data-max-stars Star Number of stars (1-10, star style only) 5
data-faces Star Number of faces (smiley style only) 3
data-version All App version for tracking —
data-show-powered-by Feedback, Helpful, Contact Show branding label true
