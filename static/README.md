# SeggWat Static Site Example

A zero-dependency HTML page that showcases all SeggWat widget types. No build tools or frameworks required.

## Usage

1. Open `index.html` in your browser
2. Replace `demo-project-key` with your actual project key from the [SeggWat dashboard](https://seggwat.com)

## What's Included

- **Feedback Button** — Floating button with screenshot support, customizable position and language
- **Helpful Rating** — Inline thumbs up/down widget
- **Star Rating** — Inline star rating (also shows smiley variant)
- **Contact Form** — Floating contact button with form modal
- **Configuration Reference** — Table of all `data-*` attributes

## Requirements

- Internet connection (widgets and CSS load from CDN)
- A SeggWat project key for actual submissions

## Customization

Edit the `<script>` tags at the bottom of `index.html` to change widget settings. See the configuration reference table on the page for all available attributes.
